## Installation

Le téléchargement du package `refpmsi` se fait à partir de Github, à
l’aide du package `devtools`

``` r
# install.packages("devtools")
# library("devtools")

devtools::install_github("denisGustin/refpmsi")
library("refpmsi")
```
